# EcoScan – Análise Inteligente de Terrenos

Uma plataforma web que analisa riscos ambientais de terrenos com base em dados simulados inspirados em fontes como INPE e IBAMA.

## Objetivo

O EcoScan foi desenvolvido com o objetivo de ajudar pessoas a tomarem decisões mais seguras antes de comprar ou investir em terrenos, identificando possíveis riscos ambientais de forma rápida e simples.

---

## Funcionalidades

- Análise de risco ambiental por localização
- Classificação de risco (baixo, médio, alto)
- Barra visual de risco
- Quiz interativo educativo
- Interface moderna e responsiva
- Animações ao rolar a página
